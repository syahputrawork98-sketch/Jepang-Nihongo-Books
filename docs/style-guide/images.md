# Style Guide Gambar

Dokumen ini menjelaskan aturan penggunaan gambar dalam repository **Japan Nihongo Books**.

---

# Tujuan Penggunaan Gambar

Gambar digunakan untuk membantu menjelaskan:

* kosakata
* objek
* situasi percakapan

Gambar harus membantu pembelajar memahami materi dengan lebih mudah.

---

# Prinsip Gambar

Gambar harus:

* jelas
* mudah dimengerti
* tidak terlalu kompleks
* fokus pada objek utama

Hindari gambar yang terlalu ramai.

---

# Prioritas Visual

Gunakan urutan berikut:

1. Emoji
2. SVG
3. PNG atau JPG

---

# Penggunaan Emoji

Gunakan emoji jika cukup jelas.

Contoh:

```
🍎 apel
📚 buku
🏫 sekolah
```

Emoji membantu membuat materi lebih ringan dan mudah dibaca.

---

# Penggunaan SVG

Gunakan SVG jika:

* membutuhkan ikon yang lebih jelas
* membutuhkan ilustrasi sederhana
* ingin menjaga ukuran file tetap kecil

---

# Format File Gambar

Gunakan format berikut:

| Format | Kegunaan                       |
| ------ | ------------------------------ |
| SVG    | ikon atau ilustrasi            |
| PNG    | gambar dengan latar transparan |
| JPG    | foto atau gambar realistis     |

---

# Lokasi Penyimpanan

Semua gambar harus disimpan di folder:

```
media/images/
```

Contoh:

```
lesson-01-perkenalan/
└── media/
    └── images/
        ├── apel.svg
        ├── guru.png
        └── sekolah.svg
```

---

# Penamaan File

Gunakan aturan berikut:

* huruf kecil
* gunakan tanda `-`
* tidak menggunakan spasi

Contoh:

```
apel.svg
guru-sekolah.png
makan-sushi.jpg
```

---

# Ukuran File

Gambar harus dioptimalkan.

Aturan umum:

* ukuran file di bawah 500 KB
* hindari resolusi terlalu besar
