# 02 - Kanji Tema Harian

## Tujuan
- Mengenali kanji dasar yang sering muncul dalam kehidupan sehari-hari.

| Kanji | Arti | Contoh | Romaji |
| --- | --- | --- | --- |
| 人 | orang | 日本人 | nihonjin |
| 日 | hari/matahari | 日曜日 | nichiyoubi |
| 月 | bulan | 月曜日 | getsuyoubi |
| 火 | api | 火曜日 | kayoubi |
| 水 | air | 水曜日 | suiyoubi |
| 木 | kayu | 木曜日 | mokuyoubi |
| 金 | uang/emas | 金曜日 | kinyoubi |
| 土 | tanah | 土曜日 | doyoubi |
| 山 | gunung | 山 | yama |
| 川 | sungai | 川 | kawa |
| 口 | mulut | 入口 | iriguchi |
| 目 | mata | 目 | me |
| 耳 | telinga | 耳 | mimi |
| 手 | tangan | 手 | te |
| 足 | kaki | 足 | ashi |

## Cara Pakai

- Pilih 5 kanji per hari.
- Hafalkan arti + 1 contoh kata.
- Baca dan tulis ulang beberapa kali.
