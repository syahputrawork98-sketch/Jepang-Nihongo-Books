# 03 - Arti Kanji Per Bulan

## Kenapa Nama Bulan Jepang Terlihat "Angka"?

Karena sistem modernnya memang literal:
- Angka + 月
- 月 berarti "bulan" (month)

Artinya:
- 一月 = bulan ke-1
- 二月 = bulan ke-2
- ...
- 十二月 = bulan ke-12

## Breakdown Per Bulan

| Bulan | Kanji | Arti Literal | Bacaan |
| --- | --- | --- | --- |
| Januari | 一月 | bulan pertama | いちがつ (ichigatsu) |
| Februari | 二月 | bulan kedua | にがつ (nigatsu) |
| Maret | 三月 | bulan ketiga | さんがつ (sangatsu) |
| April | 四月 | bulan keempat | しがつ (shigatsu) |
| Mei | 五月 | bulan kelima | ごがつ (gogatsu) |
| Juni | 六月 | bulan keenam | ろくがつ (rokugatsu) |
| Juli | 七月 | bulan ketujuh | しちがつ (shichigatsu) |
| Agustus | 八月 | bulan kedelapan | はちがつ (hachigatsu) |
| September | 九月 | bulan kesembilan | くがつ (kugatsu) |
| Oktober | 十月 | bulan kesepuluh | じゅうがつ (juugatsu) |
| November | 十一月 | bulan ke-11 | じゅういちがつ (juuichigatsu) |
| Desember | 十二月 | bulan ke-12 | じゅうにがつ (juunigatsu) |

## Catatan Penting

Untuk penggunaan standar pemula, pegang ini:
- 四月 -> しがつ
- 七月 -> しちがつ
- 九月 -> くがつ
