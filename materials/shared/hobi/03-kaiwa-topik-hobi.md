# 03 - Kaiwa Topik Hobi

A: しゅみ は なん ですか。
A: Shumi wa nan desu ka.
A: Hobi Anda apa?

B: わたし の しゅみ は おんがく です。
B: Watashi no shumi wa ongaku desu.
B: Hobi saya mendengarkan musik.

A: よく どんな おんがく を ききますか。
A: Yoku donna ongaku o kikimasu ka.
A: Anda sering mendengarkan musik seperti apa?

B: よく J-POP を ききます。
B: Yoku jei-poppu o kikimasu.
B: Saya sering mendengarkan J-POP.

A: しゅうまつ は なに を しますか。
A: Shuumatsu wa nani o shimasu ka.
A: Kalau akhir pekan biasanya ngapain?

B: ときどき えいが を みます。
B: Tokidoki eiga o mimasu.
B: Kadang-kadang saya menonton film.
