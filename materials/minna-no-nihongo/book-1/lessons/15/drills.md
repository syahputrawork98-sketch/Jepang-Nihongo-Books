## Latihan 1 - `Vて も いいです`

Lengkapi kalimat berikut.

1. ここ で しゃしん を とっ(    ) も いいです。
2. この へや に はいっ(    ) も いいです。

## Latihan 2 - `Vて は いけません`

Lengkapi kalimat berikut.

1. としょかん で おおきい こえ で はな(    ) は いけません。
2. びょういん で たばこ を すっ(    ) は いけません。

## Latihan 3 - Pertanyaan Izin

Lengkapi dialog.

1. A: ここ で けいたい を つかって も ______ ですか。
   B: はい、いいです。

2. A: ここ で たべて も いいですか。
   B: いいえ、______。
