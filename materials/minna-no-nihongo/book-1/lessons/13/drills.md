## Latihan 1 - `ほしいです`

Lengkapi kalimat berikut.

1. わたし は じてんしゃ ______ ほしいです。
2. たなかさん は カメラ ______ ほしいです。

## Latihan 2 - `～たいです`

Ubah ke bentuk `たいです`.

1. みず を のみます。 -> みず を ______。
2. えいが を みます。 -> えいが を ______。
3. にほん へ いきます。 -> にほん へ ______。

## Latihan 3 - Tanya Jawab

Lengkapi bagian kosong.

1. A: なに が ほしいですか。
   B: くるま ______ ほしいです。

2. A: どこ へ いきたいですか。
   B: きょうと へ ______。

## Latihan 4 - Susun Kalimat

Susun menjadi kalimat benar.

1. ほしいです / わたし / カメラ / は / が
2. いきたいです / うみ / へ / は / わたし
