# Japan Nihongo Books

**Japan Nihongo Books** adalah repository yang berisi kumpulan buku dan materi pembelajaran bahasa Jepang yang disusun secara terstruktur.

Repository ini dibuat sebagai **rak buku digital** untuk berbagai materi belajar bahasa Jepang seperti:

* Minna no Nihongo
* Irodori
* Materi persiapan JLPT
* Materi latihan percakapan
* Materi latihan mendengar

Struktur repository dibuat agar **rapi, konsisten, dan mudah digunakan**, baik oleh manusia maupun dengan bantuan AI.

---

# Tujuan Repository

Repository ini dibuat untuk:

* Mengorganisasi materi belajar bahasa Jepang secara terstruktur
* Menyimpan berbagai buku pembelajaran bahasa Jepang dalam satu tempat
* Memudahkan kolaborasi antara manusia dan AI
* Membuat sistem belajar yang rapi dan mudah dikembangkan

Repository ini diharapkan dapat menjadi **perpustakaan digital pembelajaran bahasa Jepang**.

# Kolaborasi

Repository ini dirancang agar bisa dikerjakan bersama oleh:

* Kontributor manusia
* Bantuan AI

Agar struktur tetap rapi dan konsisten, semua kontributor harus mengikuti aturan yang terdapat di folder **docs**.

Dokumen penting yang perlu dibaca:

* docs/STRUCTURE.md
* docs/STYLE_GUIDE.md
* docs/CONTRIBUTING.md

---

# Prinsip Proyek

Repository ini dibangun dengan beberapa prinsip utama:

**Konsisten**
Semua lesson mengikuti struktur yang sama.

**Rapi**
Penamaan folder dan file dibuat jelas dan terstruktur.

**Mudah Dikembangkan**
Buku baru dapat ditambahkan tanpa merusak struktur yang sudah ada.

**Kolaboratif**
Dirancang untuk memudahkan kerja sama antara manusia dan AI.
