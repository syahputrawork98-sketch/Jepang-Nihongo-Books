# Shared Materials

Folder ini berisi materi dasar yang dipakai bersama oleh seri Minna no Nihongo dan Irodori.

## Cakupan

- Writing system: hiragana, katakana, kanji dasar
- Grammar dasar: kata sifat (i-adjective dan na-adjective)
- Numbers: angka
- Date/Calendar: hari dan tanggal, bulan
- Money: uang
- K3-APD: dasar keselamatan kerja + kosakata Jepang terkait APD
- Hobi: kosakata dan pola percakapan dasar topik hobi

## Aturan Pakai

- Simpan materi universal di folder ini.
- Jangan duplikasi materi dasar yang sama di tiap seri.
- Di materi seri, referensikan file dari shared bila relevan.

## Struktur Saat Ini

- `writing-system/`
- `kata-sifat/`
- `numbers/` (angka)
- `hari-dan-tanggal/`
- `months/`
- `uang/`
- `K3-APD/`
- `hobi/`
