# Bulan dalam Bahasa Jepang

Modul ini dibuat untuk pemula total agar bisa paham sistem bulan Jepang dalam satu alur belajar.

## Urutan Belajar

1. [00 - Overview](./00-overview.md) (5 menit)
2. [01 - Dasar](./01-dasar.md) (15-20 menit)
3. [02 - Detail Penggunaan](./02-detail-penggunaan.md) (15-20 menit)
4. [03 - Arti Kanji Per Bulan](./03-kanji-per-bulan.md) (10-15 menit)
5. [04 - Latar Belakang Sejarah](./04-latar-belakang-sejarah.md) (5-10 menit)
6. [99 - Ringkasan](./99-ringkasan.md) (3-5 menit)

## Hasil Akhir yang Ditargetkan

Setelah menyelesaikan modul ini, kamu diharapkan bisa:
- Menyebutkan bulan 1-12 dalam bahasa Jepang.
- Menghindari kesalahan bacaan yang paling sering terjadi.
- Menggunakan kosakata bulan dalam kalimat sehari-hari.
- Memahami kenapa sistem nama bulan Jepang terlihat "angka + bulan".
