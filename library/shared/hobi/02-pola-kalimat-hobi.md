# 02 - Pola Kalimat Hobi

## Pola 1 - Menyebut Hobi

```text
わたし の しゅみ は N です。
Watashi no shumi wa N desu.
```

Arti: Hobi saya adalah N.

Contoh:
- わたし の しゅみ は どくしょ です。
- Watashi no shumi wa dokusho desu.
- Hobi saya membaca.

## Pola 2 - Menanyakan Hobi

```text
しゅみ は なん ですか。
Shumi wa nan desu ka.
```

Arti: Hobinya apa?

Contoh:
- しゅみ は なん ですか。
- Shumi wa nan desu ka.
- Hobi Anda apa?

## Pola 3 - Frekuensi Aktivitas

```text
よく N を します。 / よく Vます。
Yoku N o shimasu. / Yoku Vmasu.
```

Arti: Saya sering melakukan N / Saya sering V.

Contoh:
- よく えいが を みます。
- Yoku eiga o mimasu.
- Saya sering menonton film.

- よく スポーツ を します。
- Yoku supootsu o shimasu.
- Saya sering berolahraga.

## Pola 4 - Aktivitas Akhir Pekan

```text
しゅうまつ に N を します。
Shuumatsu ni N o shimasu.
```

Arti: Di akhir pekan saya melakukan N.

Contoh:
- しゅうまつ に テニス を します。
- Shuumatsu ni tenisu o shimasu.
- Di akhir pekan saya bermain tenis.
