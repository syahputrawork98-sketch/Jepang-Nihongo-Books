# 00 - Overview

## Kenapa Topik Hobi Penting?

Topik hobi sangat sering dipakai dalam percakapan awal:
- perkenalan
- obrolan santai
- membangun relasi

## Target Modul

- Menguasai kosakata hobi dasar.
- Menggunakan pola `しゅみ は ... です`.
- Mampu bertanya: `しゅみ は なん ですか`.
