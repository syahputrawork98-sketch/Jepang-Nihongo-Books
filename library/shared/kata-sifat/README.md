## Hasil Belajar

Setelah selesai, kamu diharapkan bisa:
- Membedakan kata sifat i dan kata sifat na.
- Membuat kalimat sederhana dengan kata sifat.
- Memakai bentuk positif dan negatif dasar.

## Catatan Cakupan

- Daftar kata sifat di modul ini adalah daftar umum untuk level dasar-menengah awal, bukan daftar kamus lengkap.
- Fokus utama modul ini adalah pemakaian praktis dalam percakapan dan kalimat sehari-hari.
- Materi dapat ditambah bertahap per tema atau per level JLPT pada pembaruan berikutnya.
