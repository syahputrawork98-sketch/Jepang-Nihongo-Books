# Irodori Rules

Dokumen ini adalah aturan seri untuk `library/irodori/`.

## Scope

- Berlaku untuk semua level di seri Irodori.
- Mengikuti aturan global `library/dok/rules.md`.

## Struktur Level

Setiap level memakai pola:

- `00-front-matter/`
- `lessons/<nn>/`
- `90-appendix/`

## Struktur Lesson

Setiap lesson wajib berisi:

- `README.md`
- `vocab.md`
- `grammar.md`
- `drills.md`
- `kaiwa.md`
- `notes.md`
- `media/audio/`
- `media/images/`

## Aturan Penulisan Konten

- Gunakan `library/dok/style-guide.md` sebagai baseline.
- Sesuaikan istilah dengan konteks Irodori.
- Gunakan lesson dua digit (`01`, `02`, dst.) agar konsisten.
