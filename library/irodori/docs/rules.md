# Irodori Rules

Dokumen ini adalah aturan seri untuk `library/irodori/`.

## Scope

- Berlaku untuk semua level di seri Irodori.
- Mengikuti aturan global `library/docs/rules.md`.

## Struktur Level

Setiap level memakai pola:

- `00-front-matter/`
- `lessons/<nn>-<slug-bab>/`
- `90-appendix/`

## Struktur Lesson

Setiap lesson wajib berisi:

- `README.md`
- `vocab.md`
- `grammar.md`
- `drills.md`
- `kaiwa.md`
- `notes.md`
- `media/audio/`
- `media/images/`

## Aturan Penulisan Konten

- Gunakan `library/docs/style-guide.md` sebagai baseline.
- Sesuaikan istilah dengan konteks Irodori.
- Gunakan pola lesson `<nn>-<slug-bab>` agar urut dan mudah dicari.
