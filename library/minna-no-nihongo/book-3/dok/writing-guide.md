# Writing Guide book-3

Dokumen ini mengatur cara menulis isi bab untuk book-3.

## Scope

- Berlaku untuk semua folder `lessons/<nn>/` pada book-3.
- Mengikuti aturan seri di `library/minna-no-nihongo/dok/rules.md`.

## Standar Isi per Lesson

Setiap lesson harus memiliki:

- `README.md`
- `vocab.md`
- `grammar.md`
- `drills.md`
- `kaiwa.md`
- `notes.md`
- `media/audio/`
- `media/images/`

## Format Penulisan

- Gunakan baseline format dari `library/dok/style-guide.md`.
- Untuk grammar, setiap pola minimal berisi:
  - pola kalimat,
  - arti,
  - contoh kalimat.

## Catatan

Aturan ini fokus pada isi bab.
Aturan rilis dan changelog tetap di level seri.
