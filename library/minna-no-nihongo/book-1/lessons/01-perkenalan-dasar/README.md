# Lesson 01 - Perkenalan Dasar

## Tujuan
- Memahami pola perkenalan diri paling dasar dalam bahasa Jepang.
- Mampu menyebut identitas sederhana (nama, asal, pekerjaan/status).
- Mampu membuat kalimat nominal dengan pola `A は B です`.

## Cakupan Materi
- Salam perkenalan dasar.
- Kosakata identitas diri.
- Pola grammar:
  - `A は B です`
  - `A は B ではありません`
  - `A は B ですか`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Memperkenalkan diri secara singkat.
- Menanyakan dan menjawab identitas lawan bicara.
- Membuat kalimat positif, negatif, dan tanya sederhana.
