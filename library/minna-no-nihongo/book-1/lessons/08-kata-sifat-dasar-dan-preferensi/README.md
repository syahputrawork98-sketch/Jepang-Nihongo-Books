# Lesson 08 - Kata Sifat Dasar dan Preferensi

## Tujuan
- Memahami penggunaan kata sifat `い` dan `な` pada kalimat dasar.
- Memahami pola preferensi `すき / きらい`.
- Memahami pola kemampuan sederhana `じょうず / へた`.

## Cakupan Materi
- Kosakata sifat dan hobi dasar.
- Pola grammar:
  - `N は い-adj です`
  - `N は な-adj です`
  - `N が すきです / きらいです`
  - `N が じょうずです / へたです`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Mendeskripsikan benda dengan kata sifat dasar.
- Menyebut suka/tidak suka terhadap hal tertentu.
- Menyebut kemampuan sederhana terhadap aktivitas.
