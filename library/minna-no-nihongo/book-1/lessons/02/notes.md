## Catatan Lesson 02

- Fokus lesson ini: kata tunjuk benda dan pertanyaan benda.
- Urutan pengajaran yang disarankan:
  1. `これ / それ / あれ`
  2. `この / その / あの + N`
  3. `なんですか`
  4. `どれ / どの N`
- Pastikan siswa paham perbedaan:
  - kata tunjuk berdiri sendiri (`これ`)
  - kata tunjuk yang wajib diikuti noun (`この ほん`)
