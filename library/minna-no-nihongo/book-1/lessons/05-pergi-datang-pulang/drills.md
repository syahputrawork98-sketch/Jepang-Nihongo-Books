# Lesson 05 - Pergi, Datang, Pulang: Drills

## Latihan 1 - Pilih Kata Kerja Perpindahan

Isi dengan: `いきます / きます / かえります`

1. わたし は がっこう へ ______。
2. ともだち が うち へ ______。
3. まいにち しちじ に うち へ ______。

## Latihan 2 - Partikel `へ` atau `で`

Isi dengan partikel yang tepat.

1. だいがく ______ いきます。
2. バス ______ いきます。
3. えき ______ かえります。

## Latihan 3 - Lengkapi Jawaban

1. だれ と いきますか。 -> ______ と いきます。
2. なに で いきますか。 -> ______ で いきます。
3. いつ いきますか。 -> ______ いきます。

## Latihan 4 - Buat Kalimat

Buat satu kalimat lengkap dari kata berikut.

1. わたし / でんしゃ / だいがく / いきます
2. サントスさん / うち / かえります
