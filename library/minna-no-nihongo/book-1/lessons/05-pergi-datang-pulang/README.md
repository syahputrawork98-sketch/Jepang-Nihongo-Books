# Lesson 05 - Pergi, Datang, Pulang

## Tujuan
- Memahami kata kerja perpindahan: `いきます / きます / かえります`.
- Memahami partikel tujuan `へ` dan tempat aktivitas `で`.
- Mampu menanyakan dan menjawab tujuan serta cara pergi.

## Cakupan Materi
- Kosakata tempat, transportasi, dan waktu.
- Pola grammar:
  - `N(Tempat) へ いきます / きます / かえります`
  - `N(Transport) で いきます`
  - `だれ と いきますか`
  - `いつ いきますか`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Menyebut ke mana pergi/datang/pulang.
- Menyebut alat transportasi yang dipakai.
- Menanyakan dengan siapa dan kapan melakukan perjalanan.
