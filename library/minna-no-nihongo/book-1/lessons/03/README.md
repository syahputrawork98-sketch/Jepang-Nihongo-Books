# Lesson 03 - Lokasi dan Tempat

## Tujuan
- Memahami kata tunjuk tempat: `ここ / そこ / あそこ`.
- Mampu menanyakan lokasi dengan pola `どこですか`.
- Memahami bentuk sopan lokasi: `こちら / そちら / あちら`.

## Cakupan Materi
- Kosakata tempat umum (sekolah/kantor/fasilitas)
- Pola grammar:
  - `ここ / そこ / あそこ は N です`
  - `N は どこ ですか`
  - `こちら / そちら / あちら`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Menjelaskan lokasi tempat secara sederhana.
- Menanyakan lokasi ruangan/fasilitas.
- Memberi arahan dasar dengan bentuk sopan.
