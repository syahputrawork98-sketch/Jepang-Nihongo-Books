# Lesson 03 - Lokasi dan Tempat: Grammar

## Pola Kalimat 1

```text
ここ / そこ / あそこ は N です。
Koko / Soko / Asoko wa N desu.
```

### Arti
Di sini/situ/sana adalah N.

### Contoh
- ここ は きょうしつ です。
- Koko wa kyoushitsu desu.
- Di sini adalah ruang kelas.

- あそこ は しょくどう です。
- Asoko wa shokudou desu.
- Di sana adalah kantin.

## Pola Kalimat 2

```text
N は どこ ですか。
N wa doko desu ka.
```

### Arti
N ada di mana?

### Contoh
- トイレ は どこ ですか。
- Toire wa doko desu ka.
- Toilet ada di mana?

- あそこ です。
- Asoko desu.
- Di sana.

## Pola Kalimat 3

```text
こちら / そちら / あちら / どちら
Kochira / Sochira / Achira / Dochira
```

### Arti
Bentuk lebih sopan untuk menunjuk lokasi/arah.

### Contoh
- うけつけ は こちら です。
- Uketsuke wa kochira desu.
- Resepsionis ada di sini.

- エレベーター は どちら ですか。
- Erebeetaa wa dochira desu ka.
- Lift ada di mana? (sopan)

## Catatan Penting

- `ここ/そこ/あそこ` adalah kata lokasi langsung.
- `こちら/そちら/あちら` lebih sopan, sering dipakai di layanan publik.
- Untuk bertanya sopan, gunakan `どちら` daripada `どこ`.
