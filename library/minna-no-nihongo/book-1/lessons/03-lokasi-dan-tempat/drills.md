# Lesson 03 - Lokasi dan Tempat: Drills

## Latihan 1 - Isi Kata Lokasi

Isi dengan: `ここ / そこ / あそこ`

1. ______ は じむしょ です。
2. ______ は しょくどう です。
3. ______ は ロビー です。

## Latihan 2 - Buat Pertanyaan Lokasi

Ubah menjadi pertanyaan `どこですか`.

1. トイレ は あそこ です。
2. うけつけ は そこ です。

## Latihan 3 - Gunakan Bentuk Sopan

Ganti kata lokasi biasa menjadi bentuk sopan.

1. ここ -> ______
2. そこ -> ______
3. あそこ -> ______
4. どこ -> ______

## Latihan 4 - Jawaban Singkat

Jawab pertanyaan berikut.

1. かいぎしつ は どこ ですか。
2. エレベーター は どちら ですか。
