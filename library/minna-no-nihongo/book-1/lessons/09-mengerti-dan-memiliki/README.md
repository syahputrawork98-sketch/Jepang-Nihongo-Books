# Lesson 09 - Mengerti dan Memiliki

## Tujuan
- Memahami kata kerja `わかります` (mengerti) dan `あります` (memiliki/ada untuk benda).
- Memahami penggunaan partikel `が` pada pola kemampuan/keberadaan.
- Mampu bertanya jenis benda dengan `どんな N`.

## Cakupan Materi
- Kosakata bahasa, hobi, dan benda sederhana.
- Pola grammar:
  - `N が わかります`
  - `N が あります`
  - `N は ありますか`
  - `どんな N`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Menyatakan mengerti atau tidak mengerti sesuatu.
- Menyatakan memiliki/tidak memiliki benda.
- Menanyakan karakter/jenis benda secara sederhana.
