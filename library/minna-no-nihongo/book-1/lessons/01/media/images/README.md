# Media Gambar - Lesson 01

Folder ini berisi gambar pendukung untuk `book-1/lessons/01`.

## Format Teknis
- Format: `svg` (disarankan) atau `jpg/png` setelah konversi
- Rasio utama: `16:9` (materi layar) + opsional `1:1` (flashcard)
- Naming: `l01-img-xx-<slug>.jpg`

## Daftar File Wajib
- `l01-img-01-perkenalan-dua-orang.svg` - Dua orang saling memperkenalkan diri
- `l01-img-02-kartu-identitas-sederhana.svg` - Nama, asal, status
- `l01-img-03-konteks-mahasiswa.svg` - Konteks `がくせい`
- `l01-img-04-konteks-karyawan.svg` - Konteks `かいしゃいん`
- `l01-img-05-peta-negara-nihon-indonesia.svg` - Ilustrasi `にほん` dan `インドネシア`
- `l01-img-06-frasa-hajimemashite.svg` - Visual frasa salam awal
- `l01-img-07-frasa-yoroshiku.svg` - Visual frasa penutup perkenalan
