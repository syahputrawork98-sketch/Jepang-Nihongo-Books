# Lesson 07 - Memberi dan Menerima

## Tujuan
- Memahami kata kerja memberi/menerima: `あげます` dan `もらいます`.
- Memahami penggunaan partikel `に`, `から`, dan `を` pada pola hadiah/barang.
- Mampu menyatakan kepada siapa memberi dan dari siapa menerima.

## Cakupan Materi
- Kosakata barang sederhana untuk hadiah.
- Pola grammar:
  - `A は B に N を あげます`
  - `A は B に/から N を もらいます`
  - `だれ に あげますか`
  - `だれ に/から もらいますか`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Menyebut siapa memberi apa kepada siapa.
- Menyebut menerima barang dari siapa.
- Membuat dialog sederhana tentang hadiah/barang.
