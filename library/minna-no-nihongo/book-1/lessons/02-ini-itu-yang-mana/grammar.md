# Lesson 02 - Ini, Itu, Yang Mana: Grammar

## Pola Kalimat 1

```text
これ / それ / あれ は N です。
Kore / Sore / Are wa N desu.
```

### Arti
Ini/itu adalah N.

### Contoh
- これ は ほん です。
- Kore wa hon desu.
- Ini adalah buku.

- それ は ペン です。
- Sore wa pen desu.
- Itu adalah pulpen.

- あれ は くるま です。
- Are wa kuruma desu.
- Itu adalah mobil.

## Pola Kalimat 2

```text
この / その / あの + N
Kono / Sono / Ano + N
```

### Arti
Ini/itu + kata benda.

### Contoh
- この かばん は わたし の です。
- Kono kaban wa watashi no desu.
- Tas ini milik saya.

- その ほん は せんせい の です。
- Sono hon wa sensei no desu.
- Buku itu milik guru.

- あの とけい は リナさん の です。
- Ano tokei wa Rina-san no desu.
- Jam itu milik Rina.

## Pola Kalimat 3

```text
N は なん ですか。
N wa nan desu ka.
```

### Arti
N itu apa?

### Contoh
- それ は なん ですか。
- Sore wa nan desu ka.
- Itu apa?

- これ は なん ですか。
- Kore wa nan desu ka.
- Ini apa?

- それ は じしょ です。
- Sore wa jisho desu.
- Itu kamus.

## Pola Kalimat 4

```text
どれ ですか。 / どの N ですか。
Dore desu ka. / Dono N desu ka.
```

### Arti
Yang mana? / Yang mana N?

### Contoh
- リナさん の かさ は どれ ですか。
- Rina-san no kasa wa dore desu ka.
- Payung Rina yang mana?

- どの かばん ですか。
- Dono kaban desu ka.
- Tas yang mana?

- どの ほん ですか。
- Dono hon desu ka.
- Buku yang mana?

## Catatan Penting

- `これ / それ / あれ` dipakai sendiri tanpa kata benda setelahnya.
- `この / その / あの / どの` wajib diikuti kata benda.
- Jarak pembicara-lawan bicara menentukan pemakaian `kore/sore/are`.
