# Lesson 02 - Ini, Itu, Yang Mana

## Tujuan
- Memahami kata tunjuk benda: `これ / それ / あれ`.
- Memahami kata tunjuk yang menerangkan kata benda: `この / その / あの`.
- Mampu menanyakan benda dengan pola `なんですか` dan `どれですか`.

## Cakupan Materi
- Kosakata benda di kelas/kantor
- Pola grammar:
  - `これ / それ / あれ は N です`
  - `この / その / あの + N`
  - `N は なんですか`
  - `どれ / どの N`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Menunjuk benda di sekitar dengan tepat.
- Menanyakan nama/jenis benda.
- Menjawab pertanyaan pilihan benda sederhana.
