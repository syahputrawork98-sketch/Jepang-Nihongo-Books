## Latihan 1 - `Vてください`

Lengkapi bentuk `て` yang tepat.

1. なまえ を か(    ) ください。
2. ドア を し(    ) ください。
3. ここ で ま(    ) ください。

## Latihan 2 - `Vて います`

Lengkapi kalimat berikut.

1. わたし は ほん を よんで ______。
2. いま、にほんご を べんきょうして ______。

## Latihan 3 - Tanya Jawab

Lengkapi dialog.

1. A: なに を していますか。
   B: しゅくだい を ______。

2. A: みなさん、しずかに して ______。
   B: はい。

## Latihan 4 - Susun Kalimat

Susun menjadi kalimat benar.

1. ください / あけて / ほん / を
2. しています / いま / わたし は / べんきょう
