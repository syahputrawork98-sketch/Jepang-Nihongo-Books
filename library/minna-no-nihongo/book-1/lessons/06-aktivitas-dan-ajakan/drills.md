# Lesson 06 - Aktivitas dan Ajakan: Drills

## Latihan 1 - Partikel Objek `を`

Lengkapi kalimat berikut dengan `を`.

1. ほん ______ よみます。
2. コーヒー ______ のみます。
3. えいが ______ みます。

## Latihan 2 - Pilih Kata Kerja yang Tepat

Isi dengan: `たべます / のみます / よみます / みます`

1. あさごはん を ______。
2. みず を ______。
3. しんぶん を ______。
4. えいが を ______。

## Latihan 3 - Ubah ke Bentuk Ajakan `ませんか`

1. いっしょに えいが を みます。
2. コーヒー を のみます。

## Latihan 4 - Ubah ke Bentuk `ましょう`

1. ひるごはん を たべます。
2. おんがく を ききます。
