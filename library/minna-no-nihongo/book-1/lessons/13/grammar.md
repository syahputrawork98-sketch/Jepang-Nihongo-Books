## Pola Kalimat 1

```text
N が ほしいです。
N ga hoshii desu.
```

### Arti
Saya ingin N (benda).

### Contoh
- カメラ が ほしいです。
- Kamera ga hoshii desu.
- Saya ingin kamera.

- くるま が ほしいです。
- Kuruma ga hoshii desu.
- Saya ingin mobil.

## Pola Kalimat 2

```text
なに が ほしいですか。
Nani ga hoshii desu ka.
```

### Arti
Ingin apa?

### Contoh
- A: なに が ほしいですか。
- B: じてんしゃ が ほしいです。

## Pola Kalimat 3

```text
Vます-stem + たいです。
```

### Arti
Ingin melakukan V.

### Contoh
- みず を のみたいです。
- Mizu o nomitai desu.
- Saya ingin minum air.

- にほん へ いきたいです。
- Nihon e ikitai desu.
- Saya ingin pergi ke Jepang.

- えいが を みたいです。
- Eiga o mitai desu.
- Saya ingin menonton film.

## Pola Kalimat 4

```text
どこ へ いきたいですか。
Doko e ikitai desu ka.
```

### Arti
Ingin pergi ke mana?

### Contoh
- A: どこ へ いきたいですか。
- B: きょうと へ いきたいです。

## Catatan Penting

- `ほしいです` dipakai untuk benda, bukan untuk verb.
- `～たいです` dipakai untuk keinginan melakukan aktivitas.
- Partikel umum pada `～たいです` tetap mengikuti verb aslinya (`を`, `へ`, dll.).
