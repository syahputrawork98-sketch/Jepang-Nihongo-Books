## Tujuan
- Memahami kata kerja memberi/menerima: `„Āā„Āí„Āĺ„Āô` dan `„āā„āČ„ĀĄ„Āĺ„Āô`.
- Memahami penggunaan partikel `„Āę`, `„Āč„āČ`, dan `„āí` pada pola hadiah/barang.
- Mampu menyatakan kepada siapa memberi dan dari siapa menerima.
