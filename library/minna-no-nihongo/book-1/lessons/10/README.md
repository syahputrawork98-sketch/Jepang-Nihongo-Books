# Lesson 10 - Lokasi Orang dan Benda

## Tujuan
- Memahami penggunaan `います` untuk orang/hewan dan `あります` untuk benda.
- Memahami kata posisi: `うえ / した / なか / そと / となり / まえ / うしろ`.
- Mampu menanyakan dan menyatakan lokasi secara sederhana.

## Cakupan Materi
- Kosakata tempat dan posisi.
- Pola grammar:
  - `N(orang/hewan) は どこ に いますか`
  - `N(orang/hewan) は [tempat] に います`
  - `N( benda ) は [tempat] に あります`
  - `N1 は N2 の [posisi] です`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Menanyakan lokasi orang/hewan dan benda.
- Menjawab lokasi dengan partikel `に`.
- Menjelaskan posisi relatif antar benda.
