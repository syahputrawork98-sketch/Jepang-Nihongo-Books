## Latihan 1 - `より`

Lengkapi kalimat berikut.

1. でんしゃ は バス ______ はやい です。
2. みかん は りんご ______ やすい です。

## Latihan 2 - `いちばん`

Lengkapi dengan `いちばん`.

1. スポーツ の なかで サッカー が ______ すき です。
2. かもく の なかで すうがく が ______ むずかしい です。

## Latihan 3 - `どちら`

Lengkapi pertanyaan berikut.

1. りんご と みかん と ______ が すき ですか。
2. コーヒー と おちゃ と ______ が やすい ですか。

## Latihan 4 - Jawaban Pilihan

Jawab pertanyaan berikut dengan `A の ほう` atau `どちら も`.

1. にほんご と えいご と どちら が むずかしい ですか。
2. コーヒー と おちゃ と どちら が すき ですか。
