# Lesson 12 - Perbandingan Dasar

## Tujuan
- Memahami pola perbandingan dengan `より`.
- Memahami superlatif sederhana dengan `いちばん`.
- Mampu menanyakan pilihan dengan `どちら`.

## Cakupan Materi
- Kosakata benda/kategori untuk perbandingan.
- Pola grammar:
  - `A は B より adj です`
  - `[kategori] の なかで N が いちばん adj です`
  - `A と B と どちら が adj ですか`
  - `どちら も ...`

## Hasil Belajar
Setelah lesson ini, kamu diharapkan bisa:
- Membandingkan dua benda secara sederhana.
- Menyebut yang paling dalam satu kategori.
- Menjawab pertanyaan pilihan dua opsi.
