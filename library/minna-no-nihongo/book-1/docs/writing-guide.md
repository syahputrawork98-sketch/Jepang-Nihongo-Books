# Writing Guide book-1

Dokumen ini mengatur cara menulis isi bab untuk book-1.

## Scope

- Berlaku untuk semua folder `lessons/<nn>-<slug-bab>/` pada book-1.
- Mengikuti aturan seri di `library/minna-no-nihongo/docs/rules.md`.

## Standar Isi per Lesson

Setiap lesson harus memiliki:

- `README.md`
- `vocab.md`
- `grammar.md`
- `drills.md`
- `kaiwa.md`
- `notes.md`
- `media/audio/`
- `media/images/`

## Baseline Operasional

- Baseline kualitas aktif mengacu ke `lesson-01`:
  - `library/minna-no-nihongo/book-1/lessons/01-perkenalan-dasar/`
- Untuk lesson 2 sampai 15, struktur dan mutu minimal mengikuti baseline ini, kecuali ada catatan pengecualian yang disepakati.

## Format Penulisan

- Gunakan baseline format dari `library/docs/style-guide.md`.
- Untuk grammar, setiap pola minimal berisi:
  - pola kalimat,
  - arti,
  - contoh kalimat.
- Gunakan judul file yang eksplisit agar mudah dipindai:
  - `# Lesson <nn> - <judul>: Kosakata`
  - `# Lesson <nn> - <judul>: Grammar`
  - `# Lesson <nn> - <judul>: Drills`
  - `# Lesson <nn> - <judul>: Kaiwa`
  - `# Lesson <nn> - <judul>: Notes`

## Checklist Kualitas

- Gunakan checklist resmi di:
  - `library/minna-no-nihongo/book-1/docs/lesson-quality-checklist.md`
- Lesson dianggap selesai jika semua item `required` terpenuhi.

## Catatan

Aturan ini fokus pada isi bab.
Aturan rilis dan changelog tetap di level seri.
